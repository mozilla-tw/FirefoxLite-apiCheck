# FirefoxLite-apiCheck
[![Bitrise build status](https://app.bitrise.io/app/84ec26cc50d05f24/status.svg?token=9GvBIw0znCi3nWAOFsOiwA&branch=master)](https://app.bitrise.io/app/84ec26cc50d05f24)

Goal
----------------

This script is our short-term goal to do healthcheck on news partner's endpoint while the long term goal is do API schema check which will merge to the certralized repo - Firefox-Lite.

Build Instructions
----------------

1. Clone the repository:

  ```shell
  git clone https://github.com/mozilla-tw/FirefoxLite-apiCheck.git
  ```
2. Since we're using submodule, run:

  ```shell
  bash apiHealthCheck.sh
  ```
